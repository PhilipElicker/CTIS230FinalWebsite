Second Collection of Images


This collection of images features liminal spaces with minimal contextual cues, which as a result, makes their settings harder to identify. These images start to blur the lines between reality and the abstract which may make them “dream-like” to some viewers.

All images were sourced from r/LiminalSpace on Reddit. Please visit this subreddit for more liminal space images.
